# **Personal Finance Tracker**

A lightweight Command Line Interface (CLI) application built with Java to help users track their income, expenses, and account balances. This project uses a PostgreSQL database to persist user data and transaction history.

* **Balance Tracking:** Real-time updates to your account balance.  
* **Transaction Management:**

  * Log Expenses.  
  * View recent transaction history.  
  * View all historical transactions.  
* **Data Persistence:** All data is stored securely in a PostgreSQL database.

**2. Create the Tables** Run these scripts to create the necessary tables:

```SQL  
    id SERIAL PRIMARY KEY,

-- Expenses/Transactions Table

    id SERIAL PRIMARY KEY,  
    user_id INT REFERENCES users(id),  
    amount DECIMAL(10, 2) NOT NULL,  
    label VARCHAR(100),  
    description TEXT,  
    date TIMESTAMP DEFAULT CURRENT_TIMESTAMP  
);
```

## **ğŸ“– Usage Guide**

Once the application is running, follow the on-screen prompts:

1. **Start:** Choose `[L]` to Log in or `[S]` to Sign up.  
2. **Dashboard:** View your current balance and the 3 most recent transactions.  
3. **Actions:**  
   * Enter `[L]` to log a new transaction.  
     * Then choose `[D]` for Deposit (adds to balance) or `[E]` for Expense (subtracts from balance).  
   * Enter `[V]` to view your full transaction history.  
   * Enter `[S]` to Sign out.
